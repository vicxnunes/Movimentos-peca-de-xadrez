# ♟️ Desenvolvendo um Simulador de Movimentos de Xadrez em C! 

🎯 Durante meus estudos em C, desenvolvi um **simulador de movimentos de xadrez**, onde é possível visualizar as movimentações das peças **Torre, Bispo, Rainha e Cavalo**.

🔹 O projeto conta com um **menu interativo**, onde o usuário pode escolher qual peça quer movimentar.  
🔹 Cada peça segue as regras do xadrez, utilizando diferentes **estruturas de repetição (`for`, `while`, `do-while`)**.  
🔹 A saída é organizada e fácil de entender, destacando cada movimento de forma clara.

## 🔥 O Que Aprendi:
✅ Uso de loops (`for`, `while`, `do-while`) para simular movimentos.  
✅ Implementação de um **menu interativo** com `switch-case`.  
✅ Como organizar a saída do terminal para melhor legibilidade.  
✅ Modularização do código para facilitar manutenção e reutilização.

## 🛠️ Tecnologias Utilizadas  

- **Linguagem:** C  
- **Biblioteca:** `stdio.h` para entrada e saída de dados
